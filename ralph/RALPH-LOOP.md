# Ralph Loop Checklist (Manual)

Use this in interactive mode to simulate the Ralph loop.

## Start of turn
- Follow RALPH-PROMPT.md and WORKFLOW.md
- Keep output concise; summarize files instead of dumping full contents
- Confirm you will run tests/benchmarks or state why you are not

## End of turn
- Summarize what changed and why
- State tests/benchmarks run (or not run)
- Ask whether to continue with the next Ralph iteration

## One-line starter prompt
Run a Ralph iteration. Follow RALPH-PROMPT.md and WORKFLOW.md. Keep output concise; summarize files instead of dumping; note tests/benchmarks run or not run. Ask for confirmation before moving to next task.
